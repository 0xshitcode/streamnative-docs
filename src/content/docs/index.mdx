## What is StreamNative?

StreamNative is a **Tauri 2** desktop app (Windows + Linux) that behaves
like Cloudstream but ships as a native binary. You install `.js`
extensions from repositories; StreamNative renders the results in a
Netflix-style shell with a built-in HLS/DASH/MP4 player and a local
HTTP proxy that neutralises CORS + anti-hotlinking.

## Why not just Cloudstream?

Cloudstream is Android-only. StreamNative is desktop-native: keyboard
shortcuts, byte-range parallel downloads, PiP-friendly window controls,
system-native file dialogs, and no APK install ceremony. The extension
format is a subset of Cloudstream's plugin API translated into
sandboxed JavaScript — most of the same providers work with a thin
port.
